NAMING STANDARDS:

- Use clear, descriptive names
- Avoid abbreviations

VARIABLES:
- camelCase (userName, totalPrice)

FUNCTIONS:
- Verb-based (getUser, createOrder, calculateTotal)

COMPONENTS:
- PascalCase (UserCard, ProductList)

FILES:
- kebab-case (user-service.js, auth-controller.ts)

BOOLEANS:
- Use is/has/should (isActive, hasAccess)

CONSTANTS:
- UPPER_CASE (MAX_USERS, API_URL)
